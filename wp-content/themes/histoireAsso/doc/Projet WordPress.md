# Projet WordPress — Session 1

> **5 sessions avant l'examen**

## Objectif

Trouve un projet WordPress (réel ou fictif) à réaliser ou refaire. Tu as jusqu'à la prochaine session pour le choisir — **deadline : 21h30**. Si tu l'as trouvé avant, tu peux commencer dès maintenant.

---

## Cadre du projet

| Critère                | Valeur          |
| ---------------------- | --------------- |
| Temps max pour trouver | 21h30           |
| Contrainte technique   | 100% WordPress  |
| Type de projet         | Réel ou fictif  |
| Développement          | Full sur mesure |

---

## Fonctionnalités obligatoires à intégrer

Les **6 fonctionnalités suivantes** doivent toutes être utilisées et visibles dans le projet final :

- ☰ **Menus de navigation** — Création et gestion des menus WordPress
- 📄 **Pages & page templates** — Pages statiques et templates personnalisés
- ⚙️ **Advanced Custom Fields (ACF)** — Champs personnalisés avec ACF
- ✉️ **Formulaire de contact** — Formulaire fonctionnel intégré au thème
- 📦 **Custom Post Type (CPT)** — Type de contenu personnalisé
- 🏷️ **Taxonomie personnalisée** — Catégories/tags propres au projet

---

## Règles importantes

1. Tout doit être développé **sur mesure, dans WordPress** — pas de constructeurs de pages type Elementor ou Divi.
2. Les 6 fonctionnalités listées doivent toutes être **utilisées et visibles** dans le projet final.
3. Le projet doit avoir du **sens fonctionnel** : un vrai besoin, même fictif (association, commerce, portfolio, événement…).

---

## Idées de projets

Voici quelques exemples pour t'inspirer :

- **Site d'une association sportive** — CPT pour les joueurs/équipes, taxonomie par catégorie/division, formulaire d'inscription, templates par section
- **Portfolio d'agence créative** — CPT pour les réalisations, taxonomie par type de projet, formulaire de contact client
- **Site événementiel local** — CPT pour les événements, taxonomie par type/lieu, formulaire d'inscription, template de détail
- **Catalogue d'une PME** — CPT pour les produits/services, taxonomie par gamme, formulaire devis

---

> 💡 **Conseil :** Pense à un projet où les Custom Post Types ont du sens (ex : équipe, services, projets, événements, recettes…). C'est souvent le point de départ qui détermine la cohérence du reste.
